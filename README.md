# Ball Blitz

**Ball Blitz** is a fast-paced 3D dodgeball game built with **Unity** and **Blender**. In this game, the player controls a single **Thrower** and competes against **three AI Dodgers**. The objective is to eliminate all opponents within **2 minutes** using precise aiming, timing, and wall ricochets in a physics-based arena.

---

## Gameplay

- **Player Role**: Thrower  
  - Aim and throw balls at AI Dodgers.  
  - Use walls for ricochets to hit dodging opponents.  
  - Eliminate all three AI Dodgers within the 2-minute limit to win.  

- **AI Opponents**: Dodgers  
  - Dodge and evade player throws using AI movement patterns.  
  - Strategically position themselves in the arena to survive.  

- **Arena**  
  - Enclosed 3D court with walls for ricochets.  
  - Dynamic physics for realistic ball movement.  

---

## Features

- Single-player gameplay against 3 AI opponents.  
- Physics-based ball mechanics with ricochets.  
- 3D arena built in Unity with assets created in Blender.  
- Timer-based challenge (2 minutes per round).  
- Simple, fast-paced, skill-based gameplay.

---

## How to Play

1. Launch the game in Unity.  
2. Control the Thrower using movement.  
3. Aim using the mouse.  
4. Try to eliminate all 3 Dodgers before the 2-minute timer ends.  

---

## Controls

- **Move**: WASD
- **Aim**: Mouse
- **Pick Up the Ball**: Press E
- **Throw Ball**: Left mouse button
